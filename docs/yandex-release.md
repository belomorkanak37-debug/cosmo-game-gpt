# Релиз под Яндекс Игры

## Финальная структура проекта

```text
/index.html
/assets/README.md
/assets/assets.json
/assets/procedural-assets.js
/js/sdk.js
/js/game.js
/js/phase4-polish.js
/docs/phase4-checklist.md
/docs/iap-catalog.json
/docs/yandex-release.md
/README.md
```

`index.html` лежит в корне проекта. Это главный файл, который должен оказаться в корне ZIP-архива.

## Как сделать ZIP

### Вариант 1 — через GitHub

1. Открой репозиторий `belomorkanak37-debug/cosmo-game-gpt`.
2. Нажми `Code` → `Download ZIP`.
3. Распакуй скачанный архив.
4. Зайди внутрь распакованной папки.
5. Выдели содержимое папки, а не саму папку:
   - `index.html`
   - `assets/`
   - `js/`
   - `docs/`
   - `README.md`
6. Заархивируй выделенные файлы в ZIP.
7. Проверь: при открытии ZIP на первом уровне должен быть `index.html`, а не папка-обёртка.

### Вариант 2 — через терминал

На macOS/Linux из корня проекта:

```bash
zip -r star-miner-yandex.zip index.html assets js docs README.md
```

На Windows можно выделить эти же элементы, нажать правой кнопкой и выбрать `Отправить` → `Сжатая ZIP-папка`.

## Что загрузить в Консоль Яндекс Игр

Загружать нужно ZIP-архив, где `index.html` находится в корне архива.

Перед модерацией проверь:

- игра открывается простым запуском `index.html`;
- в консоли браузера нет критичных JavaScript-ошибок;
- SDK подключён через `https://yandex.ru/games/sdk/v2`;
- вызывается `YaGames.init()`;
- после загрузки вызывается `ysdk.features.LoadingAPI.ready()`;
- язык берётся из `ysdk.environment.i18n.lang`;
- прогресс сохраняется через `player.setData()` для авторизованного игрока и через `localStorage` при fallback;
- rewarded-награды начисляются только в `onRewarded`;
- расходуемые инапы консумируются через `consumePurchase()` после начисления;
- fullscreen-реклама показывается только после престижа и отключается при `noads`;
- sticky-баннер можно выключить настройкой и он отключается при `noads`;
- долгий тап и системный context menu не мешают игровому полю;
- при уходе со вкладки прогресс сохраняется.

## Лидерборд

Создай лидерборд в Консоли Яндекс Игр:

- техническое имя: `star_dust`
- название RU: `Звёздная пыль`
- название EN: `Star Dust`
- тип результата: число
- сортировка: по убыванию, больше — лучше

Игра отправляет туда количество звёздной пыли.

## Remote flags

В игре поддержаны такие флаги удалённой настройки баланса:

| Флаг | Значение по умолчанию | Что меняет |
|---|---:|---|
| `oreMultiplier` | `1` | множитель базовой добычи дронов |
| `tapMultiplier` | `1` | множитель базовой силы тапа |
| `prestigeUnlockOre` | `50000` | порог руды для престижа |
| `rewardedCrystalAmount` | `12` | награда кристаллами за ежедневное rewarded-видео |

## Каталог инапов

Товары должны иметь ровно такие id, потому что эти id уже используются в коде.

| id | Тип | Название RU | Рекоменд. цена, RUB | Что выдаёт |
|---|---|---|---:|---|
| `noads` | постоянный | Убрать рекламу | 199 | отключает sticky-баннеры и fullscreen-рекламу |
| `starter_pack` | расходуемый | Стартовый набор | 99 | +120 кристаллов и +25K руды |
| `crystals_small` | расходуемый | Малая пачка кристаллов | 59 | +80 кристаллов |
| `crystals_big` | расходуемый | Большая пачка кристаллов | 249 | +450 кристаллов |
| `drill_skin_neon` | постоянный | Неоновый бур | 149 | косметический скин бура/дрона |

Расходуемые товары: `starter_pack`, `crystals_small`, `crystals_big`. Они консумируются после начисления награды.

Постоянные товары: `noads`, `drill_skin_neon`. Они проверяются через список покупок при запуске.

## Материалы карточки

Игра использует процедурную графику на Canvas, чтобы держать вес архива маленьким. Для карточки Яндекс Игр используй PNG-пак из Фазы 1:

- иконка: `512×512`;
- обложка: `1280×720`.

Если нужно заменить карточку, придерживайся стиля: яркий мультяшный космос, глубокий синий/фиолетовый фон, крупный астро-шахтёр/дрон, минимум мелких деталей.

## Финальный smoke-test

1. Открой `index.html` локально.
2. Нажми астероид 10–20 раз.
3. Купи первый апгрейд.
4. Перезагрузи страницу и проверь сохранение.
5. Открой апгрейды, ежедневку, лидерборд, настройки, магазин.
6. В окружении Яндекс Игр проверь SDK, рекламу, покупки, лидерборд и облако.
7. После проверки загрузи ZIP в Консоль и отправь на модерацию.
