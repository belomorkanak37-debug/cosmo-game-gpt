# Фаза 4 — удержание и полировка

## Что проверять

1. Оффлайн-доход
   - Зайти в игру, накопить хотя бы немного DPS.
   - Закрыть вкладку или свернуть игру больше чем на минуту.
   - Вернуться и увидеть окно оффлайн-дохода.
   - Проверить обычный сбор и кнопку x2 через rewarded-рекламу в окружении Яндекс Игр.

2. Ежедневная награда
   - При первом входе за день открывается окно ежедневной награды.
   - Стрик растёт по дням 1..7.
   - После получения в тот же день повторно не начисляется.

3. Числа и баланс
   - Валюты отображаются коротко: K, M, B и дальше.
   - DPS виден сверху.
   - Престиж становится доступен после нужного количества руды.

4. Звук
   - По умолчанию звук выключен.
   - В настройках тумблер включает лёгкие звуковые эффекты.
   - Перед рекламой игра сохраняется, геймплей ставится на паузу через SDK.

5. Мобильное поведение
   - Нет системного скроллбара.
   - Нет pull-to-refresh.
   - Нет контекстного меню по долгому тапу.
   - Кнопки крупные и нажимаются пальцем.

## Реализация

- Основная логика Фазы 4 находится в `js/game.js`.
- Дополнительные браузерные защиты находятся в `js/phase4-polish.js`.
- `index.html` подключает `phase4-polish.js` последним, после игрового ядра.
