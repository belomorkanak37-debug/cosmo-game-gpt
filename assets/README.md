# Ассеты

Фаза 1: визуальный набор для игры «Звёздный Старатель».

Стиль: яркий мультяшный космос, глубокий синий и фиолетовый фон, неоновые акценты, крупные читаемые формы для мобильных экранов.

Состав: фон, астероиды, дроны, валюты, UI-иконки, кнопки, панели и эффекты.

Файл procedural-assets.js создаёт ассеты на Canvas прямо в браузере, чтобы проект оставался лёгким и запускался без сборщика.
