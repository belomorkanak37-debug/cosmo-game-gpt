# Звёздный Старатель

HTML5 idle-игра для Яндекс Игр: космический шахтёр, добыча руды, дроны, апгрейды и престиж через прыжки в новые галактики.

## Статус разработки

- Фаза 1: ассеты и визуальный стиль — готово.
- Фаза 2: ядро игры — готово к проверке.
- Фаза 3: SDK Яндекс Игр — дальше.
- Фаза 4: удержание и полировка — дальше.
- Фаза 5: сборка и инструкция публикации — дальше.

## Структура

- `index.html` — корневая точка входа для Яндекс Игр.
- `assets/` — визуальный слой и манифест ассетов.
- `js/game.js` — игровое ядро, баланс CONFIG, сохранения, апгрейды, престиж, достижения и ежедневная награда.

## Как запустить локально

Откройте `index.html` в браузере. Игра не требует сборщика.
