# Звёздный Старатель

HTML5 idle-игра для Яндекс Игр: космический шахтёр, добыча руды, дроны, апгрейды и престиж через прыжки в новые галактики.

## Статус разработки

- Фаза 1: ассеты и визуальный стиль — готово.
- Фаза 2: ядро игры — готово.
- Фаза 3: интеграции SDK Яндекс Игр — готово.
- Фаза 4: удержание и полировка — готово.
- Фаза 5: сборка и инструкция публикации — готово к проверке.

## Структура

- `index.html` — корневая точка входа для Яндекс Игр.
- `assets/` — визуальный слой и манифест ассетов.
- `js/sdk.js` — безопасный слой SDK Яндекс Игр: игрок, облачные сохранения, реклама, покупки, лидерборды, флаги, review и Gameplay/Loading API.
- `js/game.js` — игровое ядро, баланс CONFIG, сохранения, апгрейды, престиж, достижения, ежедневная награда и SDK-точки входа.
- `js/phase4-polish.js` — дополнительная мобильная полировка: защита от overscroll, context menu и резервный marker выхода.
- `docs/phase4-checklist.md` — чеклист проверки удержания и полировки.
- `docs/iap-catalog.json` — список инапов и лидерборда для Консоли Яндекс Игр.
- `docs/yandex-release.md` — инструкция сборки ZIP, загрузки и smoke-test перед модерацией.

## Как запустить локально

Откройте `index.html` в браузере. Вне Яндекс Игр SDK может быть недоступен — игра перейдёт в локальный режим через `localStorage`.

## Как собрать ZIP

Заархивируйте из корня проекта: `index.html`, `assets`, `js`, `docs`, `README.md`.

Важно: в корне ZIP должен лежать `index.html`, а не папка-обёртка.
